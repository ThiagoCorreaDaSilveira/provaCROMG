o Projeto tem por objetivo:

  Permitir ao usuário o cadastro de perfil, estabelecendo acesso à plataforma
  Permitir ao usuário CRUD de meta dados de "Filmes"
  Permitir ao usuário CRUD de meta dados de "Endereços"
  Apresentar knowhow técnico para avaliação de cadandidato à vaga de desenvolvimento PHP 
  
O projeto foi desenvolvido em:
  PHP 7.2
  Laravel 5.6
  Base de Dados SQLite (gerada via Migration)
  Apache2 Server

Testado e Desenvolvido em:
  Linux - Ubuntu 18.04.1
  Netbeans 8
  DB Browser for SQLite
  Google Chrome

Pré Requisitos (tal como solicitado em documentação Laravel 5.6):
  PHP >= 7.1.3
  OpenSSL PHP Extension
  PDO PHP Extension
  Mbstring PHP Extension
  Tokenizer PHP Extension
  XML PHP Extension
  Ctype PHP Extension
  JSON PHP Extension
  
  Necessário definir base URL em:
    config/app.php ou .env ('APP_URL', 'http://localhost/provaCROMG/public/')
